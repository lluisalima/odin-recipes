# Odin Recipes 🍽️

This project was built as part of the Foundations course from The Odin Project.

It is a simple recipe website created using basic HTML concepts, including:

* HTML boilerplate
* Headings and paragraphs
* Links between pages
* Images
* Lists
* Folder organization

## Recipes Included

* Banana Bread
* Carbonara
* Lemon Pie

## Project Goal

The purpose of this project was to practice fundamental HTML skills by building a multi-page recipe website.

## Technologies Used

* HTML5

## What I Learned

Through this project, I practiced:

* Creating and organizing HTML files
* Using relative links
* Adding images to webpages
* Structuring content with lists and headings
* Basic project organization

Built with ❤️ during my journey through The Odin Project.
